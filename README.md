# Классификация дефектов микрополосковой линии

Проект для моделирования микрополосковой линии с дефектами, генерации синтетических данных измерений и классификации типов дефектов с помощью случайного леса.

## Структура проекта

- `config/` – конфигурационные файлы YAML
- `data/` – сгенерированные CSV-данные
- `models/` – сохранённые модели
- `reports/` – текстовые отчёты и изображения
- `src/` – исходный код (пакет)
- `scripts/` – исполняемые скрипты
